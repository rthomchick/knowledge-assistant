# PM Knowledge Assistant

A RAG-powered knowledge base for ServiceNow product management.

## What it does
Upload your team's documents and ask questions in plain English.
Answers are grounded in your actual documents — no hallucinated facts,
no generic responses.

## How to use
1. Upload documents (txt or markdown) in the sidebar
2. Select the document type for better filtering
3. Ask questions and get cited answers

## Stack
- Claude Sonnet for reasoning
- OpenAI text-embedding-3-small for embeddings  
- ChromaDB for vector storage
- Streamlit for the interface

## Built during
6-month AI Product Development Program — Week 6: RAG & Knowledge-Grounded AI
